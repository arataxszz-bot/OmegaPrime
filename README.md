# Omega Prime AI Commander

This is a Java Mindustry mod project. It is not a raw Android-importable source folder, because Android is not secretly a Gradle build machine just waiting to be appreciated.

## The practical way to run it on Android

1. Put this project in a GitHub repository from your phone.
2. Wait for GitHub Actions to build it.
3. Download the artifact from the Actions tab.
4. Unzip the artifact and take the `.jar` inside.
5. Put that `.jar` into Mindustry's `mods` folder.
6. Open Mindustry, go to Mods, and enable Omega Prime.

## What this project includes

- Commander AI scaffold
- Drone swarm system
- Adaptive threat learning
- Overmind tactical panel
- Research and save systems
- Mobile-first HUD launcher

## Notes

- This project uses the standard Mindustry Java mod shape.
- Mindustry Java mods are built as `.jar` files and can be loaded on desktop and Android once compiled.
- GitHub Actions is the easiest build path if you only have a phone.
